# GUI-based-spectrometer
This is a python based spectrometer that can run on current Raspberry Pi OS. 
You can execute "GUI_spectrum_vX_X.py" then very shimple GUI will come out. Jelly-jiro has been never trained for any programming, and this software was coded only by github copilot on Visual Studio Code. Thanks alot for AI.

In order to use this software, some packages would be required. Please refer on README and install them before running the software. Both en and jp version are available.

[Hardware requirement]
This software use following items.
1. Monochromator
    --Telescope based monochromator that can puachase on Amazon easily.
2. Raspberry py camera with C-mount
    --In order to focus inside monochrometer sufficiently, c-mount equipped model looks better to add lens.
